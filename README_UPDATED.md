   pip install colored
   pip install colorama
   pip install pystyle
   pip install requests termcolor PySocks  # опционально

     },
     "smtp": {...},
     "domains": {...}
   }

ПАРАМЕТРЫ ОТПРАВКИ: data/sending_config.json
──────────────────────────────────────────
   Управляется через: python3 NewLinks.py → Меню 5

   Параметры:
   • senders_per_batch: 30-500+ (количество отправителей за раз)
   • min_delay_between_emails: задержка между письмами (сек)
   • max_delay_between_emails: максимальная задержка
   • rotate_proxy_every_n_emails: смена прокси каждые N писем

КАСТОМНЫЕ ПРОКСИ: data/custom_proxies.json
────────────────────────────────────────
   Создается автоматически при добавлении прокси:
   python3 NewLinks.py → Меню 4 → Добавить прокси

ОТПРАВИТЕЛИ: data/ukraine_senders_1200.json
──────────────────────────────────────────
   1200 украинских email:password пар
   
   Генерируется:
   python3 ukraine_senders_1200.py
