# .CreateSNOS-TG

Краткое описание
-----------------
Это утилитный репозиторий для генерации и отправки наборов данных и вспомогательных скриптов (генераторы ссылок, конфигурации рассылки, вспомогательные проверки). Проект содержит набор Python-скриптов и конфигурационных файлов для подготовки и работы с данными в папке `data/`.

Структура (корень)
-------------------
- `config/` — настройки и шаблоны конфигураций
- `data/` — входные и выходные JSON-файлы (senders, receivers и др.)
- `scripts/` — вспомогательные утилиты (например, `scripts/auto_push.sh`)
- Основные скрипты: `START_HERE.py`, `QUICKSTART.py`, `proxy_generator.py`, `crypto_email_gen.py`, `NewLinks.py`.

Быстрая установка
-----------------
1. Клонируйте репозиторий.
2. Создайте и активируйте виртуальное окружение Python.
3. Установите зависимости:
```bash
pip install -r requirements.txt
```

Как запустить
-------------
- Для справки запустите `python START_HERE.py` или откройте `QUICKSTART.py`.
- Для одиночных утилит запускайте соответствующие скрипты, например:
```bash
python NewLinks.py
python proxy_generator.py
```
