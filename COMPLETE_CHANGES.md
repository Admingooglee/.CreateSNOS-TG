  • 1200 email:password пар для отправки
  • Создается: python3 ukraine_senders_1200.py

data/custom_proxies.json
  • Ваши пользовательские прокси
  • Создается: когда добавляешь прокси через меню 4
  • Формат: {"proxies": ["http://1.2.3.4:8080", ...]}

data/sending_config.json
  • Параметры отправки (batch_size, задержки и т.д.)
  • Создается: при запуске sending_config.py или NewLinks.py
  • Формат: JSON с объектом batch_settings, timing, proxy_settings

logs/snoser.log
  • Логи отправок
  • Обновляется: при каждой отправке
  • Формат: "timestamp - LOGGER - LEVEL - message"
